# Refrigerator

**Attributes:**

1. brandName (string)

2. isOpen (boolean)

3. color (string)

4. currentTemp (float)

5. maxTemp (integer)

6. magnets (array)

**Methods**

1. closeDoor (modifies isOpen boolean attribute)

2. addMagnet (adds magnet into the magnet array)

3. changeColor (modifies color string attribute)

4. changeTemp (modifies currentTemp by 5.0)
